# Tastenkuerzel

Lorus Musik Macher bietet umfangreiche Tastenkuerzel fuer effizientes Arbeiten. Alle Shortcuts koennen in den [[Einstellungen]] angepasst werden.

## Uebersicht anzeigen

Druecke **Shift+?** um die Tastenkuerzel-Uebersicht in der App anzuzeigen.

## Standard-Tastenkuerzel

### Wiedergabe

| Aktion | Tastenkuerzel |
|--------|--------------|
| Play / Pause | `Space` |
| Vorspulen | `ArrowRight` |
| Zurueckspulen | `ArrowLeft` |
| Zufaelliger Track | `R` |

### Lautstaerke

| Aktion | Tastenkuerzel |
|--------|--------------|
| Lauter | `ArrowUp` |
| Leiser | `ArrowDown` |

### Geschwindigkeit

| Aktion | Tastenkuerzel |
|--------|--------------|
| Schneller | `Ctrl+ArrowUp` |
| Langsamer | `Ctrl+ArrowDown` |
| Normalgeschwindigkeit | `Ctrl+0` |

### Tonhoehe (Pitch)

| Aktion | Tastenkuerzel |
|--------|--------------|
| Tonhoehe hoch (+1 Halbton) | `Shift+ArrowUp` |
| Tonhoehe runter (-1 Halbton) | `Shift+ArrowDown` |
| Tonhoehe zuruecksetzen | `Shift+0` |

### Cue-Points

| Aktion | Tastenkuerzel |
|--------|--------------|
| Cue-Point 1-9 springen | `1` - `9` |
| Cue-Point 1-9 setzen | `Shift+1` - `Shift+9` |

### Panels & Features

| Aktion | Tastenkuerzel |
|--------|--------------|
| A-B Loop umschalten | `B` |
| Equalizer ein/aus | `E` |
| Warteschlange oeffnen | `Q` |
| Lyrics oeffnen | `L` |
| Suche oeffnen | `Ctrl+F` |
| Favorit umschalten | `F` |

### Bearbeitung

| Aktion | Tastenkuerzel |
|--------|--------------|
| Rueckgaengig | `Ctrl+Z` |
| Wiederherstellen | `Ctrl+Shift+Z` |

### Sonstiges

| Aktion | Tastenkuerzel |
|--------|--------------|
| Tastenkuerzel-Uebersicht | `Shift+?` |

## Tastenkuerzel anpassen

### Recording-Modus

1. Oeffne die [[Einstellungen]]
2. Scrolle zum Abschnitt **Tastenkuerzel**
3. Klicke auf den Button neben der gewuenschten Aktion
4. Der Recording-Modus startet â€” die Taste blinkt
5. Druecke die gewuenschte Tastenkombination
6. Die neue Taste wird zugewiesen

### Modifier-Tasten

Folgende Modifier koennen kombiniert werden:

| Modifier | Taste |
|----------|-------|
| Ctrl | Steuerungstaste |
| Shift | Umschalttaste |
| Alt | Alt-Taste |

Beispiele: `Ctrl+S`, `Shift+A`, `Ctrl+Shift+Z`

### Standard-Shortcuts wiederherstellen

1. Oeffne die [[Einstellungen]]
2. Scrolle zum Abschnitt **Tastenkuerzel**
3. Klicke auf **Zuruecksetzen**
4. Alle Shortcuts werden auf die Standardwerte zurueckgesetzt

## Hinweise

- Shortcuts funktionieren nur, wenn kein Eingabefeld fokussiert ist
- Die Gross-/Kleinschreibung wird beim Matching ignoriert
- Modifier-Kombinationen muessen exakt uebereinstimmen
- Einige Shortcuts (wie `Ctrl+F`) haben Browser-Standardverhalten, das von der App ueberschrieben wird
