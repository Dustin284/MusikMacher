# Erste Schritte

Diese Anleitung fuehrt durch die ersten Schritte mit Lorus Musik Macher.

## Uebersicht der Oberflaeche

Die App ist in mehrere Bereiche aufgeteilt:

- **Seitenleiste (links)** — Bibliotheken, Tags, Navigation
- **Hauptbereich (mitte)** — Track-Tabelle mit Such- und Filteroptionen
- **Player (unten)** — Waveform, Playback-Steuerung, Cue-Points
- **Panels (rechts)** — Warteschlange, Lyrics, FX (ein-/ausblendbar)
- **Tabs (oben)** — Browse, Import, Premiere Pro, Statistik, Einstellungen

## Schritt 1: Musik importieren

### Dateien und Ordner importieren

1. Wechsle zum **Import**-Tab
2. Klicke auf **Ordner auswaehlen** oder **Dateien auswaehlen**
3. Waehle die gewuenschten Audio-Dateien (MP3, WAV, M4A, OGG, FLAC, WebM)
4. Waehle die Ziel-Bibliothek (z.B. "Songs" oder "Effekte")
5. Klicke auf **Importieren**

**Tipp:** Wenn du einen Ordner mit Unterordnern importierst, werden die Unterordner-Namen automatisch als Tags zugewiesen.

### Aus dem Internet herunterladen

1. Wechsle zum **Import**-Tab
2. Fuege einen YouTube-, SoundCloud- oder Spotify-Link ein
3. Klicke auf **Herunterladen**
4. Der Track wird automatisch heruntergeladen und importiert

Mehr dazu unter [[Import und Download]].

## Schritt 2: Projekt erstellen (optional)

Wenn du Tracks nach Projekten organisieren moechtest:

1. Klicke unter "Projekte" in der Seitenleiste auf **Neues Projekt**
2. Vergib einen Projektnamen
3. Waehle das Projekt aus — importierte Tracks werden automatisch zugeordnet
4. Wechsle zu "Alle Projekte", um alle Tracks zu sehen

**Tipp:** Wenn du einen Track aus einem Projekt entfernen moechtest, oeffne das Rechtsklick-Menue und waehle "Aus Projekt entfernen". Der Track bleibt unter "Alle Projekte" erhalten.

## Schritt 3: Bibliotheken organisieren (optional)

Die App startet mit zwei Standard-Bibliotheken:

- **Songs** — Fuer Musiktitel
- **Effekte** — Fuer Sound-Effekte

### Eigene Bibliothek erstellen

1. Klicke auf das **+**-Icon neben "Bibliotheken" in der Seitenleiste
2. Gib einen Namen ein und waehle ein Icon
3. Die neue Bibliothek erscheint in der Seitenleiste

Die Standard-Bibliotheken (Songs, Effekte) koennen nicht geloescht werden.

## Schritt 4: Tracks abspielen

1. Klicke auf einen Track in der Tabelle, um ihn auszuwaehlen
2. Doppelklick oder **Space** startet die Wiedergabe
3. Nutze die Waveform zum Navigieren (Klick auf eine Position)
4. Steuerung ueber [[Tastenkuerzel]] oder die Player-Buttons

## Schritt 5: Tracks organisieren

### Bewertung

- Klicke auf die Sterne in der Track-Zeile (1-5 Sterne)
- Oder per Rechtsklick > Bewerten

### Tags zuweisen

- Rechtsklick auf einen Track > Tags zuweisen
- Oder ueber die Tag-Seitenleiste Tags erstellen und zuweisen

### Favoriten

- Klicke auf das Herz-Icon neben einem Track
- Oder druecke **F** bei ausgewaehltem Track
- Nutze den Favoriten-Filter in der Toolbar, um nur Favoriten anzuzeigen

### Kommentare

- Rechtsklick auf einen Track > Kommentar bearbeiten
- Kommentare werden in der Tabelle angezeigt (Spalte muss sichtbar sein)

## Schritt 6: Spalten anpassen

1. Klicke auf das Zahnrad-Icon in der Track-Tabelle
2. Waehle, welche Spalten angezeigt werden sollen:
   - Dauer, BPM, Tonart, Bewertung, Tags, Kommentar
3. Die Einstellung wird gespeichert

## Schritt 7: Tags erstellen

1. Oeffne die Tag-Seitenleiste
2. Klicke auf **Tag erstellen**
3. Gib einen Namen ein
4. Tags koennen als Favorit markiert oder versteckt werden

### Smart Tags

Smart Tags sind regelbasierte Tags, die automatisch zugewiesen werden:

- BPM-Bereich (z.B. 120-140 BPM)
- Tonart
- Bewertung
- Text im Name oder Kommentar
- Dauer
